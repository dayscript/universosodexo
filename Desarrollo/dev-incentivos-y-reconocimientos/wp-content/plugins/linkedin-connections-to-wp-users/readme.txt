== Description ==

This plugin helps any WordPress administrator to add all of thier LinkedIn connections as WordPress subscribers of the site.
He just have to export the CSV from their LinkedIn profile and import it to WordPress dashboard using this plugin. 

Only will user Name and Email from your LinkedIn connections, Allow you to send Emails on users creation with thier Username and Passwords

 <ul>
 <li> Import LinkedIn Connections CSV as Wordpress Users.</li>
 <li> Will Create Username and password.</li> 
 <li> Email Notification can be controlled.</li>
 <li> User Role will be subscirbers</li>
 <li> Easy to use.</li>
 </ul>

More Options coming soon.
 

== Installation ==

This section describes how to install the plugin and get it working. 

<ol>
<li>Just login to the wordpress admin.</li>
<li>Goto plugins / Upload the Plugin folder to wp-content/plguins folder</li>
<li>Install new Plugin</li>
<li>Upload a plugin </li>
<li>Activate.</li>
<li>You will see a new tab 'Import Linkedin Connections'</li>
</ol>
